<h2>About the Project</h2>
<p>
  <strong>Travelling</strong> is a responsive website built using HTML, CSS, and JavaScript.  
  It is dedicated to the theme of travel and includes an attractive image slider showcasing the best destinations.  
  The project is fully responsive, so it looks great on both desktop and mobile devices.
</p>

<h3>Features:</h3>
<ul>
  <li>🌍 Travel-related content with a clean and modern design</li>
  <li>📸 Image slider implemented using JavaScript</li>
  <li>📱 Fully responsive layout for mobile devices</li>
</ul>
